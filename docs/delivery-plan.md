# Ten-Week Delivery Plan

| Week | Focus | Gate |
|---|---|---|
| 1 | Repository, architecture, consent, scheduling | CI and local stack run |
| 2 | Recorder, ingestion, session review | Recording is replayable |
| 3 | SOP generation and editor, market test 1 | Approved SOP in under 60s |
| 4 | Interactive onboarding | New user follows approved SOP |
| 5 | Voice feedback and classification | Feedback links to SOP steps |
| 6 | Market test 2 and pricing | User completes task unassisted |
| 7 | Timing and path analytics | Multiple sessions compare reliably |
| 8 | Dashboard and evidence-backed summary | Insights link to evidence |
| 9 | Market test 3 and security review | Analytics under 5s for 50 sessions |
| 10 | Integration, documentation, pitch, handoff | Client completes core flow |

## Six-Person Ownership

| Person | Primary | Secondary |
|---|---|---|
| 1 | Product, customer tests, requirements | UX research, pitch |
| 2 | Extension and recorder | Security, recorder tests |
| 3 | Web app and onboarding | UI/UX |
| 4 | API, persistence, authentication | Docker and deployment |
| 5 | SOP, transcription, AI adapters | Prompt evaluation |
| 6 | Analytics and automated tests | DevOps and documentation |

## Weekly Cadence

- Monday: plan and confirm ownership.
- Wednesday: integrate and remove blockers.
- Friday: demonstrate the complete flow, not isolated modules.
- Feature freeze begins after Week 8.
